버튼은 3가지만 확실히 구분하세요

초보자가 가장 많이 흔들리는 부분이 버튼입니다. "주요 1개(Elevated) + 보조(Outlined) + 가벼운(Text)"만 엄격히 지켜도 화면이 정돈됩니다.

1) ElevatedButton (주요 CTA)

hover/active/disabled 상태를 직접 테스트해보세요.

Flutter 코드
ElevatedButton(
  onPressed: () {},
  style: ElevatedButton.styleFrom(
    minimumSize: const Size.fromHeight(48),
    shape: RoundedRectangleBorder(
      borderRadius: BorderRadius.circular(8),
    ),
  ),
  child: const Row(
    mainAxisSize: MainAxisSize.min,
    children: [
      Icon(Icons.check),
      SizedBox(width: 8),
      Text('확인'),
    ],
  ),
);
2) OutlinedButton (보조)
Flutter 코드
OutlinedButton(
  onPressed: () {},
  style: OutlinedButton.styleFrom(
    minimumSize: const Size.fromHeight(48),
    side: const BorderSide(width: 2),
    shape: RoundedRectangleBorder(
      borderRadius: BorderRadius.circular(8),
    ),
  ),
  child: const Text('취소'),
);
3) TextButton (가벼운 액션)
Flutter 코드
TextButton(
  onPressed: () {},
  style: TextButton.styleFrom(
    minimumSize: const Size.fromHeight(40),
    shape: RoundedRectangleBorder(
      borderRadius: BorderRadius.circular(8),
    ),
  ),
  child: const Text('건너뛰기'),
);
4) IconButton
Flutter 코드
IconButton(
  onPressed: () {},
  icon: const Icon(Icons.favorite),
  tooltip: '좋아요',
);
5) FloatingActionButton (FAB)

화면의 "대표" 행동을 한 번에 보여주는 버튼입니다. 남발하면 오히려 혼란스럽습니다.

Flutter 코드
FloatingActionButton(
  onPressed: () {},
  child: const Icon(Icons.add),
);

FloatingActionButton(
  mini: true,
  onPressed: () {},
  child: const Icon(Icons.edit),
);
