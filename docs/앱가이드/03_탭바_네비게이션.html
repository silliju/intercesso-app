1) TabBar (상단 탭)

탭은 "같은 화면 안에서 카테고리만 바뀌는" 경우에 적합합니다. (예: 기도/감사 피드)

기도 피드
기도 글 목록이 여기에 표시됩니다.
감사 피드
감사 글 목록이 여기에 표시됩니다.
저장됨
북마크한 글이 여기에 표시됩니다.
Flutter 코드 (DefaultTabController + TabBar)
DefaultTabController(
  length: 3,
  child: Scaffold(
    appBar: AppBar(
      title: const Text('피드'),
      bottom: const TabBar(
        tabs: [
          Tab(text: '기도 피드'),
          Tab(text: '감사 피드'),
          Tab(text: '저장됨'),
        ],
      ),
    ),
    body: const TabBarView(
      children: [
        PrayerFeedPage(),
        ThanksFeedPage(),
        SavedPage(),
      ],
    ),
  ),
);
2) BottomNavigationBar (하단 네비)

앱의 큰 섹션(홈/기도/기록/감사/알림)을 이동할 때 사용합니다. 아래는 클릭하면 선택 상태가 바뀌는 데모입니다.

설명
웹 페이지이므로 실제 라우팅은 하지 않고, active 상태만 바뀝니다.
Flutter 코드 (BottomNavigationBar)
int currentIndex = 0;

Scaffold(
  body: pages[currentIndex],
  bottomNavigationBar: BottomNavigationBar(
    currentIndex: currentIndex,
    onTap: (index) => setState(() => currentIndex = index),
    type: BottomNavigationBarType.fixed,
    items: const [
      BottomNavigationBarItem(icon: Icon(Icons.home), label: '홈'),
      BottomNavigationBarItem(icon: Icon(Icons.book), label: '기도'),
      BottomNavigationBarItem(icon: Icon(Icons.history), label: '로그'),
      BottomNavigationBarItem(icon: Icon(Icons.favorite), label: '감사'),
      BottomNavigationBarItem(icon: Icon(Icons.notifications), label: '알림'),
    ],
  ),
);
