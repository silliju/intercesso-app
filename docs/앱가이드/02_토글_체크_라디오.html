1) Switch (설정 ON/OFF)

아래 토글을 클릭해보세요. 오른쪽 상태 텍스트가 바뀝니다.

상태: ON
상태: OFF
Flutter 코드 (Switch / SwitchListTile)
bool isNotificationOn = true;

SwitchListTile(
  title: const Text('알림 받기'),
  value: isNotificationOn,
  onChanged: (value) {
    setState(() => isNotificationOn = value);
  },
);

Switch(
  value: isNotificationOn,
  onChanged: (value) {
    setState(() => isNotificationOn = value);
  },
);
2) Checkbox (복수 선택)
선택됨: 이용약관
Flutter 코드 (CheckboxListTile)
bool agreeTerms = true;
bool agreeMarketing = false;

CheckboxListTile(
  title: const Text('이용약관 동의'),
  value: agreeTerms,
  onChanged: (value) {
    setState(() => agreeTerms = value ?? false);
  },
);

CheckboxListTile(
  title: const Text('마케팅 수신(선택)'),
  value: agreeMarketing,
  onChanged: (value) {
    setState(() => agreeMarketing = value ?? false);
  },
);
3) Radio (단일 선택)

결제수단처럼 "하나만" 선택해야 할 때 사용합니다.

선택됨: card
Flutter 코드 (RadioListTile)
String payment = 'card';

RadioListTile(
  title: const Text('신용카드'),
  value: 'card',
  groupValue: payment,
  onChanged: (value) {
    setState(() => payment = value!);
  },
);

RadioListTile(
  title: const Text('계좌이체'),
  value: 'bank',
  groupValue: payment,
  onChanged: (value) {
    setState(() => payment = value!);
  },
);
