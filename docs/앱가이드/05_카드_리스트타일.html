1) 카드(Card) 스타일 3종

카드는 "정보를 묶는 상자"입니다. 앱 전체에서 radius/그림자 규칙을 고정하면 화면이 매우 안정적으로 보입니다.

기본 카드(Default)
테두리 + 그림자가 있는 기본 카드입니다. 피드 아이템에 적합합니다.
person 참여 3 chat_bubble 댓글 12
플랫 카드(Flat)
배경만 살짝 달라서 구역을 나눌 때 좋습니다. (그림자 없음)
아웃라인 카드(Outline)
강조하고 싶지만 elevation을 올리고 싶지 않을 때 테두리로만 강조합니다.
Flutter 코드 (Card + shape + elevation)
Card(
  elevation: 2,
  shape: RoundedRectangleBorder(
    borderRadius: BorderRadius.circular(12),
    side: const BorderSide(color: Color(0xFFE9ECEF)),
  ),
  child: Padding(
    padding: const EdgeInsets.all(24),
    child: Column(
      crossAxisAlignment: CrossAxisAlignment.start,
      children: const [
        Text('기본 카드', style: TextStyle(fontSize: 18, fontWeight: FontWeight.w700)),
        SizedBox(height: 8),
        Text('카드 내용...'),
      ],
    ),
  ),
);
2) ListTile (설정/목록 화면의 기본)

ListTile은 "좌측 아이콘 + 제목/설명 + 우측 보조" 패턴입니다. 설정 화면, 리스트 화면에서 매우 자주 쓰입니다.

person
프로필
이름/사진/소개를 수정합니다
chevron_right
notifications
알림 설정
푸시/이메일 알림을 관리합니다
chevron_right
lock
관리자 메뉴(비활성)
권한이 없으면 접근할 수 없습니다
block
Flutter 코드 (ListTile)
ListTile(
  leading: const Icon(Icons.notifications),
  title: const Text('알림 설정'),
  subtitle: const Text('푸시/이메일 알림을 관리합니다'),
  trailing: const Icon(Icons.chevron_right),
  onTap: () {},
);
