폼은 "에러 상태"까지 설계해야 완성입니다

초보자 실수: 입력은 만들었는데, 에러 메시지/포커스/disabled가 없어서 앱이 허술해 보입니다.

1) 로그인 폼 데모(검증 포함)
이메일
형식: name@domain.com
올바른 이메일 형식이 아닙니다.
비밀번호
영문/숫자 조합을 권장합니다.
비밀번호는 8자 이상이어야 합니다.
Flutter 코드 (TextFormField + validator)
final _formKey = GlobalKey();

Form(
  key: _formKey,
  child: Column(
    children: [
      TextFormField(
        decoration: const InputDecoration(
          labelText: '이메일',
          hintText: 'example@email.com',
        ),
        validator: (v) {
          if (v == null || v.isEmpty) return '이메일을 입력하세요';
          final ok = RegExp(r'^[^@]+@[^@]+\.[^@]+$').hasMatch(v);
          return ok ? null : '올바른 이메일 형식이 아닙니다';
        },
      ),
      const SizedBox(height: 16),
      TextFormField(
        obscureText: true,
        decoration: const InputDecoration(
          labelText: '비밀번호',
          hintText: '8자 이상',
        ),
        validator: (v) {
          if (v == null || v.length < 8) return '비밀번호는 8자 이상';
          return null;
        },
      ),
      const SizedBox(height: 24),
      ElevatedButton(
        onPressed: () {
          if (_formKey.currentState!.validate()) {
            // 통과
          }
        },
        child: const Text('로그인'),
      ),
    ],
  ),
);
2) 다양한 입력 형태
검색
메모(멀티라인)
비활성 입력
인증번호(OTP) 예시
각 칸에 1자리씩 입력하고 자동으로 다음 칸으로 이동합니다.
Flutter 코드 (OTP - TextField 4개)
Widget otpBox({required FocusNode node, FocusNode? next}) {
  return SizedBox(
    width: 44,
    child: TextField(
      focusNode: node,
      textAlign: TextAlign.center,
      keyboardType: TextInputType.number,
      maxLength: 1,
      decoration: const InputDecoration(counterText: ''),
      onChanged: (v) {
        if (v.length == 1 && next != null) {
          FocusScope.of(context).requestFocus(next);
        }
      },
    ),
  );
}
