1) Slider

슬라이더는 볼륨/밝기 같은 연속 값을 조절할 때 사용합니다. 아래는 움직일 때 값과 프로그레스가 같이 변합니다.

볼륨50
밝기70
Flutter 코드 (Slider)
double volume = 50;

Slider(
  value: volume,
  min: 0,
  max: 100,
  divisions: 100,
  label: volume.round().toString(),
  onChanged: (v) => setState(() => volume = v),
);
2) LinearProgressIndicator
업로드40%
버튼으로 진행률을 바꿔보세요
무한 로딩(Indeterminate)
Flutter 코드 (LinearProgressIndicator)
// 결정형(진행률 값이 있는 경우)
LinearProgressIndicator(value: progress); // progress: 0.0 ~ 1.0

// 무한 로딩(값이 없으면 자동 애니메이션)
const LinearProgressIndicator();
