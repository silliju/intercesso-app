나의 기도를 기록하고, 공동체와 함께 기도하는 플랫폼

  • 중보기도 — 누군가를 위해 대신 기도해 드려요
  • 작정기도 — 7·21·40·100일 목표 기도를 도전하세요

시작하기

회원가입부터 로그인까지 — 3분이면 충분해요

📝 회원가입

1

앱 실행 → 온보딩 화면

앱을 처음 열면 Intercesso를 소개하는 3장의 화면이 나타납니다. 오른쪽으로 넘기거나 건너뛰기를 눌러 로그인 화면으로 이동하세요.

2

회원가입 버튼 탭

로그인 화면 하단의 "계정이 없으신가요? 회원가입"을 누릅니다.

3

정보 입력

아래 항목을 차례로 입력하세요.

항목설명필수여부
📧 이메일로그인에 사용할 이메일 주소필수
🔒 비밀번호6자 이상 (영문+숫자 권장)필수
✏️ 닉네임공동체에서 표시될 이름필수
⛪ 교회명소속 교회 (검색 편의)선택
💡 닉네임은 나중에 프로필 수정에서 바꿀 수 있어요.
4

가입 완료 → 홈으로 이동

회원가입 버튼을 누르면 자동으로 로그인되어 홈 화면으로 이동합니다. 🎉

🔑 로그인

1

이메일 & 비밀번호 입력

가입 시 등록한 이메일과 비밀번호를 입력하세요.

2

비밀번호 찾기

비밀번호가 기억나지 않으면 "비밀번호 찾기"를 눌러 이메일로 재설정 링크를 받으세요.

3

자동 로그인

한 번 로그인하면 앱을 다시 열어도 자동으로 홈 화면이 열립니다. 로그아웃 전까지 다시 입력할 필요 없어요.

🔐 보안을 위해 공용 기기에서는 사용 후 반드시 로그아웃해 주세요.

홈 화면

기도 현황을 한눈에 — 대시보드와 최근 기도

📊 나의 기도 현황 카드

홈 상단에 나의 기도 통계가 한눈에 보입니다.

항목설명
총 기도지금까지 작성한 기도 수
응답받음상태를 "응답받음"으로 바꾼 기도 수
🔥 연속 기도일매일 기도 참여를 이어온 일수
참여 기도다른 분의 기도에 함께한 횟수
응답률(응답받음 ÷ 총 기도) × 100%

📋 최근 기도 목록

  • 최신 공개 기도 최대 5개가 표시됩니다.
  • 기도 카드를 탭하면 기도 상세 화면으로 이동해요.
  • 전체 보기 → 를 누르면 기도 목록 탭으로 이동합니다.
  • 공개 기도가 없을 경우 나의 기도 목록이 대신 표시됩니다.
✅
홈 화면은 아래로 당기면(Pull-to-Refresh) 최신 정보로 새로고침됩니다.

기도 목록 & 기도 작성

나의 기도를 기록하고, 공동체의 기도를 함께해요

📂 기도 목록 — 탭 구분

상단 4개의 탭으로 원하는 기도를 골라 볼 수 있어요.

탭보여주는 기도
🌐 전체 공개모든 사용자의 공개 기도
✏️ 내 기도내가 작성한 기도 전체
🤝 지인 기도나를 팔로우하거나 그룹 안 지인의 기도
🙏 기도중내가 "기도 참여"한 기도들

🏷️ 카테고리 필터

탭 아래 가로 스크롤 칩으로 카테고리를 선택하면 해당 기도만 보입니다.

전체 💊 건강 👪 가정 🎯 진로 ✝️ 영적 💼 사업 📌 기타

🔍 기도 검색

화면 상단 검색 아이콘을 누르면 검색창이 열립니다. 기도 제목이나 내용의 키워드를 입력하면 실시간으로 필터링됩니다.

ℹ️
검색은 현재 탭의 기도 목록 안에서만 동작합니다. 전체 탭에서 검색하면 공개 기도 전체를 찾아줘요.

✍️ 기도 작성하기

화면 우측 하단의 ➕ 버튼을 눌러 새 기도를 작성합니다.

1

제목 입력 (필수)

기도의 제목을 간결하게 적어요. 최대 100자.

2

내용 입력 (필수)

기도 내용을 자세히 적어요. 구체적일수록 함께 기도하는 분들이 더 잘 이해할 수 있어요.

3

카테고리 선택 (선택)

건강 / 가정 / 진로 / 영적 / 사업 / 기타 중 하나를 선택하세요. 탭하면 선택 해제도 됩니다.

4

공개 범위 설정

범위공개 대상
🌐 전체 공개모든 Intercesso 사용자
👥 지인 공개나를 아는 사람들만
🏘 공동체같은 그룹 멤버만
🔒 비공개나만 볼 수 있어요
5

작정기도 설정 (선택)

작정기도 토글을 켜면 기간(7 / 21 / 40 / 50 / 100일)을 선택할 수 있어요. 매일 체크인으로 진도를 기록합니다.

🔥 작정기도는 꾸준한 기도 습관을 만드는 데 큰 도움이 됩니다!
6

완료 버튼 탭

우측 상단 "완료"를 누르면 기도가 저장됩니다. 🙏

📖 기도 상세 화면

기도 카드를 탭하면 상세 화면이 열립니다. 할 수 있는 것들:

👁️ 기도 참여

하단 "기도하기 🙏" 버튼을 탭하면 함께 기도하는 목록에 추가됩니다. 다시 탭하면 참여 취소.

💬 댓글

위로를 전하거나 기도 응답을 나눠요. 내가 쓴 댓글은 길게 누르면 삭제 옵션이 뜹니다.

📝 상태 변경 (내 기도)

🙏 기도중아직 기도 중
✅ 응답받음기도가 응답됐어요!
🙌 감사감사의 마음을 나눠요

🗑️ 삭제 (내 기도)

우측 상단 메뉴 → 삭제. 확인 다이얼로그가 뜹니다.

🔥 작정기도 체크인

작정기도로 설정한 기도의 상세 화면에는 체크인 섹션이 나타납니다.

1

진행률 확인

상단 프로그레스 바에서 전체 기간 대비 체크인 횟수를 한눈에 볼 수 있어요.

2

오늘 체크인

"오늘 기도했어요 ✅" 버튼을 눌러 오늘의 기도를 기록합니다. 하루에 한 번만 가능해요.

💡 이미 체크인한 날은 버튼이 비활성화되며 "오늘 완료 ✔"로 표시됩니다.
⚠️
작정기도 체크인은 본인(기도 작성자)만 가능합니다.
🤝

중보기도

나를 위해 기도해 줄 분을 요청하거나, 다른 분의 기도를 도와드려요

📤 중보기도 요청하기

기도 상세 화면에서 "중보기도 요청하기" 버튼을 탭합니다. (내 기도 + 기도중 상태일 때만 표시)

요청 유형설명
🌐 전체 공개Intercesso 사용자 모두에게 요청
👥 그룹내가 속한 특정 그룹에 요청
🧑 개인특정 사람을 직접 지정해 요청

요청 시 한마디 메시지를 함께 보낼 수 있어요.

📥 중보기도 요청 받기

누군가 나에게 중보기도를 요청하면 하단 탭 🤝 중보에 알림 배지가 생깁니다.

1

받은 요청 탭 확인

중보기도 화면의 "받은 요청" 탭에 목록이 표시됩니다.

2

수락 또는 거절

각 카드의 수락 ✅ / 거절 ❌ 버튼을 눌러 응답하세요. 수락하면 기도 상세로 바로 이동해 함께 기도할 수 있습니다.

📋 보낸 요청 확인

"보낸 요청" 탭에서 내가 보낸 중보기도 요청의 상태를 확인할 수 있어요.

⏳ 대기중 ✅ 수락됨 ❌ 거절됨

그룹

교회, 셀, 소모임, 가족과 함께 기도하는 공간

🔍 그룹 찾기 & 참여

1

초대코드로 참여

그룹 화면 우측 상단 🔑 키 아이콘을 탭합니다. 그룹 관리자에게 받은 초대코드를 입력하면 바로 가입됩니다.

💡 초대코드는 영문+숫자 조합 6자리예요.

➕ 그룹 만들기

1

그룹 화면 → ➕ 아이콘 탭

그룹 목록 우측 상단의 + 버튼을 누릅니다.

2

그룹 정보 입력

항목설명
그룹명필수, 그룹의 이름
설명선택, 그룹 소개
그룹 유형교회 / 셀·구역 / 소모임 / 가족
공개 여부공개 ON → 누구나 참여 가능
3

그룹 만들기 버튼

완성하면 그룹 만들기를 탭. 자동으로 관리자(어드민)가 됩니다.

📋 그룹 상세 화면

그룹을 탭하면 2개 탭이 있습니다.

🙏 기도 탭

  • 그룹 전용 기도 목록을 볼 수 있어요
  • 우측 하단 ➕ 버튼으로 그룹 기도 작성
  • 아래로 당기면 새로고침

👥 멤버 탭

  • 그룹 멤버 목록과 역할(어드민) 확인
  • 어드민은 상단 초대코드 복사 버튼으로 코드를 공유 가능
ℹ️
관리자(어드민) 배지가 있는 멤버가 그룹을 운영합니다. 어드민만 초대코드를 확인·공유할 수 있어요.

검색은 현재 탭에 로드된 기도 중에서만 작동합니다. 검색 전에 먼저 원하는 탭(예: 전체 공개)을 선택하고, 목록이 완전히 로드된 후 검색 아이콘을 탭해 키워드를 입력하세요. 네트워크 상태가 나쁘면 로딩이 완료되지 않아 검색 결과가 적게 나올 수 있어요.
